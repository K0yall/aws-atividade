Servidor Web AWS EC2

Hospedado na nuvem com Apache2 e VPC personalizada

Sobre o Projeto

Este projeto faz parte da atividade de redes na nuvem. A estrutura foi provisionada dentro da plataforma AWS (Amazon Web Services), utilizando uma VPC customizada com sub-redes públicas e privadas.

Infraestrutura

A máquina virtual (instância EC2) está configurada na subnet pública, acessível pela internet através do Internet Gateway. O serviço Apache2 está instalado e servindo esta página.

Objetivo

Demonstrar a criação e funcionamento de um ambiente básico na nuvem AWS, com acesso público a um servidor web funcional, confirmando o entendimento dos componentes de rede e hospedagem.
